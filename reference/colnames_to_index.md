# Get the Index for the Specified Column Names

Get the Index for the Specified Column Names

## Usage

``` r
colnames_to_index(colnames, columns)
```

## Arguments

- colnames:

  (chr) Vector of column names of a `data.frame` from which the index /
  position needs to be found.

- columns:

  (chr/int) The names or indices of the columns whose indices in the
  `colnames`-vector should be returned.

## Value

integer - the position of `columns` in `colnames`
